# Codenames

Пример многопользовательской игры Codenames, адаптированной под подготовку к интервью (слова на поле — JS/TS/Front-end концепты). В этом примере показаны возможности интеграции LLM: например, AI в роли капитана (Spymaster) для игры в одиночку или проверка ответов в фазе Check.
