##Self-assesment
В разработке использовались:
- TypeScript
- фреймворк Angular
- автоматические тесты
- библиотека Taiga UI
Были реализованы компоненты:
1. Страница Main c реализацией смены состояний: для зарегистрированного и незарегистрированного пользователя. Внедрена локализация (русский и английский язык) с помощью Transloco. Подключена смена тем. На странице есть анимация, реализована с помощью директивы Eye Compass (вращающиеся глаза котиков следом за мышкой). На странице размещены карточки игр, настроен роутинг для перехода на страницы игр.
2. Страница 404. Подключена к проекту, реализован переход на главную страницу с помощью кнопки, настроен роутинг.
3. Игра Hangman Game ("Виселица"). Для игры внедрена локализация (частично), вопросы и ответы на английском языке. Игра реализована в виде трех экранов: Menu, Game и Results. На странице Menu есть навигация с помощью кнопок. На странице Game есть описание игры, клавиатура для ввода ответа, поле вопроса и анимация для ошибок. Реализована логика игры "Виселица". Пользователь вводит ответ с помощью клавиатуры, нажатые клавиши подсвечиваются (верно или неверно). При проигрыше или прохождении пользователь видит модальное окно с кнопкой перехода к следующему или возвращения к текущему заданию. На странице результатов реализована статистика игры, наглядно показана в виде шкал прохождения каждого уровня. Результаты сохраняются в Local storage.
