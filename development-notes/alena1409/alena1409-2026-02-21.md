### Моя идея проекта

Если будет выбран формат каталога, я планирую реализовать игру: Word Merge Associations.

**Игра на ассоциации**

### Концепция:

- Игроку даётся пул карточек со словами.

- Необходимо собрать группу из 4 слов по ассоциативному признаку.

- После правильной сборки:

  - слова объединяются в одну плашку с категорией,

  - появляется блок с вопросами (от 1 до 5 — настраивается), вопросы относятся к собранной категории.


### Планируемый функционал:

- Реализация механики Drag and Drop.
- Динамическое формирование категорий.
- Блок с вопросами по собранной группе.
- Блок добавления/редактирования слов:
  - либо как задача для backend ,
  - либо самостоятельная реализация через Firebase.
 - Проверка ответов через AI-агента (если получится реализовать).

### Почему я выбрала эту игру:

- В рамках одной игры можно реализовать несколько компонентов.
- Есть возможность интеграции AI.
- Задача выходят за пределы моей зоны комфорта  и знаний (Firebase, AI).

### Возможные сложности:

- Работа с Firebase:
  - нет устойчивого понимания архитектуры,
  - много нюансов, в которых легко запутаться.

- Интеграция AI:
  - есть опасение не справиться с логикой,
  - неопределённость в реализации проверки ответов.

### Возможные решения:

- Использование чата для обучения работы c AI и, возможно, быстрой навигации по возможностям Firebase.
- Пошаговая реализация логики игры (разбить на мелкие задачи). Сначала реализовать базовую механику без AI, затем усложнять.
