Отдельно подумала над задачей от лида: системой ачивок и достижений 🏆

Сейчас я вывожу количество правильных ответов и неправильных и мод(с ИИ или без) при завершении игры.

По обсуждению с чатом выделили такие возможные варианты:

**Счёт (score)** - основная метрика 📊

- количество правильных ответов
- бонусы за серию правильных ответов (combo) 🔥
- штраф за ошибки ❌

**Точность (accuracy)** - процент правильных ответов 🎯

- `correct / total`
- можно использовать для ачивок (например: 100%, 90%, 75%)
- лучше учитывать только при достаточном количестве вопросов

**Количество игр** 📅

- сколько раз пользователь играл
- для ачивок вроде «Новичок», «Опытный игрок»

**Серии (streak / combo)** 🔥

- максимальное количество правильных ответов подряд
- награда за стабильность

**Смешанная модель** ⚖️

- score - для прогресса
- accuracy - для качества
- total games - для опыта
- combo - для вовлечённости

Планирую послушать на собрании, какие идеи у команды по поводу их игр, и уже потом определить, что имено буду реализовывать у себя. Не хочется возиться с датами и временем точно.
